# <kbd>module</kbd> `dummy_zmq_np.py`
ZMQ Publisher for streaming simulated or real neural data. 

This script sets up ZMQ publishers to stream neural data and spike events. It can generate sine wave data, use manually created data, or load data from files.

## <kbd>function</kbd> `get_sine_wave`

```python
get_sine_wave(n_channels=1024, fs=20000, total_time_steps=600000)
```

Generate a sine wave dataset. 



**Args:**
 
 - <b>`n_channels`</b> (int):  Number of channels. Default is 1024. 
 - <b>`fs`</b> (int):  Sampling frequency in Hz. Default is 20000. 
 - <b>`total_time_steps`</b> (int):  Total number of time steps. Default is 20000*30 (30 seconds). 



**Returns:**
 
 - <b>`numpy.ndarray`</b>:  2D array of sine wave data with shape (n_channels, total_time_steps).

## <kbd>function</kbd> `run`

```python
run(data_path, random_events=True)
```

Run the ZMQ publisher to stream neural data and spike events. 



**Args:**
 
 - <b>`data_path`</b> (str):  Path to the data file or 'sine' for generated sine wave or 'manual' for manually created data. 
 - <b>`random_events`</b> (bool):  Whether to generate random spike events. Default is True.

 - <b>`NotImplementedError`</b>:  If the data file type is not supported.
